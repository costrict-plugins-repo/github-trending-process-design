# /design-workflow
Design how work flows through the team.
## Steps
1. **Map and redesign** — Define stages, criteria, hand-offs using `workflow-design` skill.
2. **Set oversight** — Add just-enough governance using `governance-model` skill.
3. **Design touchpoints** — Define the supporting rituals using `ritual-design` skill.
## Output
A workflow design: staged flow with entry/exit criteria, a light governance model, and a ritual set.
Consider following up with `/scale-program` once the workflow is proven.
